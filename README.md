Jak wgrać wtyczkę:

1. Kliknij 3 kropki / paski w prawym górnym rogu przeglądarki
2. Kliknij w "Rozszerzenia"
3. Włącz "Tryb Dewelopera"
4. Kliknij "Załaduj rozpakowane"
5. Wybierz folder w którym znajduje się wtyczka
6. Włącz wtyczkę
7. Gotowe!
